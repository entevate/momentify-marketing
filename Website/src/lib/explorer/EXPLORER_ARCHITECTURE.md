# Explorer Builder Architecture

## Three-Layer Separation

### 1. LAYOUT (never changes)
**File:** `src/components/explorer/explorer.css`
**Status:** LOCKED v1.0.0 (2026-04-07)

This file defines ALL positioning, sizing, flexbox/grid layout, spacing, glassmorphic effects, typography scale, icon weights, animations, and component structure. It was pixel-matched to the Cat Defense and Freeman HTML prototypes.

**Rules:**
- DO NOT edit this file for new templates
- DO NOT add or remove CSS rules
- DO NOT change any numeric values (padding, margin, gap, width, height, font-size, border-radius)
- DO NOT change flexbox/grid properties
- Every color reference uses `var(--exp-*)` CSS variables, never hardcoded hex/rgba

**Fixed dimensions:**
- Shell: 1366x1024px
- Inner: 911x683px @ 1.5x scale
- Top bar padding: 20px 40px
- Bottom bar padding: 20px 40px
- Trait header height: 135px
- Trait grid: 4 columns, 14px gap
- Result grid: 3-col (small), 2-col (medium), 1-col (large)
- Cards per page: 6 (small), 2 (medium), 1 (large)

---

### 2. BRAND (changes per client)
**Files:**
- `src/lib/explorer/theme.ts` — generates `--exp-*` CSS variables from config
- `src/lib/explorer/types.ts` — TypeScript types for BrandingConfig

**What's configurable per brand:**

| Property | Config Path | CSS Variable |
|---|---|---|
| Primary accent | `branding.colors.primary` | `--exp-cyan` |
| Teal/brand color | `branding.colors.teal` | `--exp-teal` |
| Background gradient | `branding.colors.dark.bgGradient` | `--exp-bg-gradient` |
| Surface color | `branding.colors.dark.surface` | `--exp-surface` |
| Text colors | `branding.colors.dark.text1/2/3` | `--exp-text-1/2/3` |
| Border color | `branding.colors.dark.border` | `--exp-border` |
| Input styling | `branding.colors.dark.inputBg/Text/Placeholder` | `--exp-input-*` |
| CTA gradient | `branding.ctaGradient` | `--exp-cta-gradient` |
| CTA text color | `branding.ctaTextColor` | `--exp-cta-text` |
| Gradient word | `branding.gradientWord` | `--exp-gradient-word` |
| Font family | `branding.font` | `--exp-font` |
| Logo (dark/light) | `branding.logo.dark / .light` | (img src, not CSS) |
| Aurora orbs | `branding.auroraOrbs.orb1/2/3` | (inline style) |
| Role backgrounds | `branding.roleBackgrounds` | (inline style) |

**Auto-derived variables (from teal/primary):**
- `--exp-teal-10` (10% opacity)
- `--exp-teal-30` (30% opacity)
- `--exp-teal-40` (40% opacity)
- `--exp-cyan-15` (15% opacity)
- `--exp-card-active-border`, `--exp-card-active-bg`, `--exp-card-saved-bg`

### 3. CONTENT (changes per experience)
**File:** `src/lib/explorer/configs/*.ts` (one per client/experience)

**What's configurable per experience:**

| Property | Config Path | Description |
|---|---|---|
| Steps | `steps[]` | Ordered array of step configs (splash, registration, trait-selection, results, summary, content-library, thank-you) |
| Content cards | `content[]` | Array of ContentCard objects with title, description, icon, cardType, mediaType, stat |
| Registration | `registration` | Fields, modes (scan/form/search), labels, skip behavior |
| Features | `features` | Toggle notes, voice capture, media capture, capture info |
| Step titles | `steps[].title / .subtitle` | Per-step heading text |
| Trait options | `steps[].options[]` | Role/interest cards with value, label, icon |
| Tab config | `steps[].tabs[]` | Results tab labels, icons, filters |
| Cards per page | `steps[].cardsPerPage` | Override default (6) for small view |

## Creating a New Template

1. **Copy** an existing config (e.g., `configs/clarium.ts`)
2. **Change** only the branding colors, logo paths, and content
3. **DO NOT** touch `explorer.css` or any component TSX files
4. **Test** by loading the new config in the explorer page

Example minimal config change for a new brand:
```typescript
const NEW_BRAND: ExplorerConfig = {
  ...CLARIUM_CONFIG,
  id: 'new-brand',
  name: 'New Brand Explorer',
  branding: {
    ...CLARIUM_CONFIG.branding,
    logo: { dark: '/new-brand-dark.svg', light: '/new-brand-light.svg' },
    colors: {
      ...CLARIUM_CONFIG.branding.colors,
      primary: '#FF6B35',  // your accent color
      teal: '#FF6B35',     // your accent color
    },
    ctaGradient: 'linear-gradient(135deg, #FF6B35, #E84D0E)',
    ctaTextColor: '#FFFFFF',
  },
  steps: [ /* your steps */ ],
  content: [ /* your content cards */ ],
};
```
